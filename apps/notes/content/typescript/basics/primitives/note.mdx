## 核心理解

基础原始类型描述的是值属于哪一大类，比如字符串、数字、布尔值和空值。

```ts
let name: string = 'front notes'
let count: number = 1
let ready: boolean = true
```

它和字面量类型的区别是：`string` 可以是任意字符串，`'pending'` 只能是这一个具体字符串。
