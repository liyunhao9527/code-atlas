## 核心问题

泛型解决的是：逻辑可以复用，但类型不应该丢失。

```ts
function wrap<T>(value: T) {
  return { value }
}
```

调用 `wrap('hello')` 后，返回值里的 `value` 仍然是 `string`，而不是模糊的 `unknown`。
