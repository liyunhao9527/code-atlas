## 现在这套结构的目标

让不同技术主题各自成长，但共享抽象和学习资产。

新的主线里：

- `apps/notes/content/` 写 MDX 笔记、文章 demo 和示例代码
- 文章专属 demo 放在对应文章目录里
- `apps/notes/` 用 Astro 渲染内容站

## 重点边界

应用负责展示和路由，内容负责知识组织，demo 负责交互实验。
