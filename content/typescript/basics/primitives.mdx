## 写笔记建议

不要只列出类型名称，更值得写的是哪些场景会让类型变得含糊。

## 最小例子

```ts
let name: string = 'front notes'
let count: number = 1
let ready: boolean = true
```

基础类型本身不难，真正要记录的是它们和接口、表单、接口返回值结合时的边界。
