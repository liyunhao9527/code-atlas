## 核心理解

字面量类型不是说“这个值大概是字符串”，而是说“这个值只能是某一个具体字符串”。

```ts
type ButtonSize = 'small' | 'medium' | 'large'
type OrderStatus = 'pending' | 'paid' | 'shipped' | 'cancelled'
```

它通常会和联合类型一起出现，用来表达一组选项、状态、角色或者模式。
