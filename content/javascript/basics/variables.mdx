## 建议记录什么

重点记住变量声明方式会怎样影响可见范围、提升行为和重复声明。

- 为什么尽量优先用 `const`
- `let` 和块级作用域的关系
- `var` 为什么容易制造意外

## 最小观察

```js
if (true) {
  var legacyName = 'var'
  let blockName = 'let'
}

console.log(legacyName)
console.log(blockName)
```

这段代码适合用来验证：`var` 更像函数级作用域，`let` 和 `const` 会被限制在块级作用域里。
